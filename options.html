🦋 X to Bluesky — 設定

Bluesky の認証情報を入力してください。
App Password を使用します（メインパスワードは使用しません）。

例: yourname.bsky.social
Bluesky → 設定 → プライバシーとセキュリティ → アプリパスワード で発行できます
セキュリティについて
• 認証情報は chrome.storage.local に保存され、この拡張のみがアクセスします。
• 外部サーバーへの送信は一切行いません。通信先は bsky.social のみです。
• アプリパスワードは Bluesky の 設定 → プライバシーとセキュリティ → アプリパスワード からいつでも無効化できます。

投稿オプション

引用元ポストの X リンクをテキスト末尾に追加します
YouTube リンクを含むポストにサムネイル付きカードを自動添付します

X の DOM 構造が変更された場合にセレクタを更新できます。
