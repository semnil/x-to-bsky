# プライバシーポリシー — X to Bluesky Crossposter

最終更新日: 2026-04-05

## 概要

X to Bluesky Crossposter は、X (Twitter) の投稿を Bluesky に同時投稿する Chrome 拡張機能です。本ポリシーでは、拡張機能が扱うデータの種類、利用目的、送信先について説明します。

## 収集するデータと目的

### 認証情報

- **対象**: Bluesky ハンドルおよびアプリパスワード
- **目的**: Bluesky API (bsky.social) での認証にのみ使用
- **保存先**: デバイス上の `chrome.storage.local` にローカル保存。bsky.social 以外のサーバーには一切送信されません

### ユーザーアクティビティ

- **対象**: X (Twitter) で投稿ボタンを押した時点のテキストおよび画像
- **目的**: bsky.social に送信し、対応する Bluesky 投稿を作成
- **保存**: 投稿処理完了後、拡張機能内には保存されません

### ウェブサイトコンテンツ（オプション）

- **対象**: 投稿に含まれる URL の OGP メタデータ（ページタイトル、説明文、サムネイル画像）
- **目的**: Bluesky 上のリンクカードプレビューの生成
- **条件**: 設定画面で「リンクカード」機能を明示的に有効化した場合のみ。デフォルトでは無効
- **保存**: 保存されません。メタデータは取得後、bsky.social にリンクカードとして送信され、破棄されます

## 収集しないデータ

- 閲覧履歴、アナリティクス、テレメトリは**収集しません**
- X やその他のサイトでの閲覧ページは**追跡しません**
- 拡張機能の開発者はユーザーのデータを**受信・保存・アクセスしません**
- データの販売、第三者との共有、広告目的での利用は**一切行いません**

## データの送信先

| 送信先 | 送信データ | 目的 |
|---|---|---|
| `bsky.social` / `*.bsky.network` | ハンドル、アプリパスワード、投稿テキスト、画像 | 認証および投稿作成 |
| 投稿中の URL（オプション） | HTTP GET リクエスト（認証情報なし） | リンクカード用 OGP メタデータの取得 |

上記以外の外部サーバーへの通信は行いません。

## データの保存とセキュリティ

- すべての認証情報と設定は `chrome.storage.local` に保存され、この拡張機能のみがアクセス可能です
- デバイス間での同期やクラウドへの保存は行いません
- アプリパスワードは Bluesky の 設定 → プライバシーとセキュリティ → アプリパスワード からいつでも無効化できます
- 拡張機能をアンインストールすると、ローカルに保存されたすべてのデータが削除されます

## 権限

- **storage**: 認証情報と設定をローカルに保存
- **host_permissions** (`bsky.social`, `*.bsky.network`): Bluesky API との通信
- **optional_host_permissions** (`<all_urls>`): リンクカード有効化時にのみ実行時に要求。機能を無効化すると自動的に解除されます

## 第三者依存

なし。外部ライブラリ、SDK、CDN、アナリティクスツールは一切使用していません。

## ポリシーの変更

変更がある場合は更新日を改訂の上、本ページに掲載します。変更後も拡張機能を継続使用した場合、改訂後のポリシーに同意したものとみなします。
